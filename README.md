# Phishing Detection with Deep Learning

## Project Description
Binary classification of phishing vs legitimate emails using two deep learning approaches.

## Models
1. TF-IDF + Feedforward Neural Network
2. Embedding + LSTM

## Dataset
Kaggle Phishing Emails Dataset:
https://www.kaggle.com/datasets/subhajournal/phishingemails

Dataset is not included in this repository.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Place dataset in /data folder
3. Run notebook or training script
